# 🎓 IIMM Ayodhya College Website

A responsive and modern frontend website created for IIMM Ayodhya College using HTML, CSS, and Bootstrap.

---

## 🚀 Features

- Responsive Design (Mobile + Desktop Friendly)
- Clean and Modern UI
- Multiple Pages:
  - Home Page
  - About Page
  - Contact Page
  - Login Page
- Bootstrap Components (Navbar, Cards, Forms, etc.)

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
